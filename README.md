# Mini Bank System

## Описание

Mini Bank System — веб-приложение для просмотра клиентов банка, их счетов и банковских операций.

Проект разработан в рамках тестового задания с использованием Django REST Framework и React.

## Используемые технологии

### Backend

* Python
* Django
* Django REST Framework
* PostgreSQL

### Frontend

* React
* React Router
* Axios

## Реализованный функционал

### Главная страница

Отображает:

* количество клиентов;
* количество счетов;
* количество операций;
* общий баланс по всем счетам.

### Клиенты

Список клиентов банка с возможностью перехода в карточку клиента.

### Карточка клиента

Отображает:

* ФИО;
* ИИН;
* телефон;
* список банковских счетов клиента.

### Страница счета

Отображает:

* номер счета;
* валюту;
* баланс;
* историю операций.

Дополнительно реализованы:

* поиск по описанию операции;
* фильтрация по типу операции;
* сортировка по дате;
* пагинация.

### Аналитика

Отображает:

* количество клиентов;
* количество счетов;
* количество операций;
* общий баланс.

### Страница 404

Отображает сообщение «Страница не найдена».

## Запуск Backend

```bash
cd backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

Backend:

http://127.0.0.1:8000/

## Запуск Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend:

http://localhost:5173/

## База данных

В проекте используется PostgreSQL.

## Тестовые данные

Создано:

* 10 клиентов;
* более 20 банковских счетов;
* более 200 операций.
